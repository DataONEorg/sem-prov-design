LTER Example metadata
===============================

This folder is is used to collect example files from researchers participating in the LTER project. 


test_corpusA_v1 
  - contains metacat-style 45 docids from 3 example LTER sites.
  - Metadata are not yet in dataone. 
  - To download the metadata, please replace the docid in this link:
http://tropical.lternet.edu/knb/metacat?action=read&qformat=xml&sessionid=&docid=knb-lter-sbc.1001


test_corpus_B.txt:
  - contains 130 D1 member node identifiers from 14 LTER sites
  - datasets from test_corpus_A are included (but with permanent id)
